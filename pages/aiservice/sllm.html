Technical Pillars

베이직인터내셔널의 독보적인 7가지 핵심 기술 모듈

01

지식 행정 인텔리전스 수집

범정부 데이터를 실시간 동기화하고 행정 지식의 복잡한 맥락을 벡터화하여 RAG 답변 품질을 극대화합니다.

하이브리드 법률 크롤러

국가법령정보센터, 정부24, 보도자료를 API 및 셀레니움 기반으로 정밀 수집합니다.

계층 구조 청킹

법령의 [편-장-절-조] 위계 구조를 보존하여 검색 시 의미론적 정확도를 확보합니다.

02

고정밀 파싱 및 비식별화

비정형 행정 문서의 표, 이미지 등 시각적 요소를 정형 데이터로 변환하고 민감 개인정보를 철저히 차단합니다.

Dual-Path OCR

로컬 GPU 기반 비전 모델을 사용하여 복잡한 HWP 표 구조를 99% 이상 인식합니다.

로컬 실시간 비식별화

외부 통신 없이 로컬 sLLM 기반으로 이름, 연락처, 주민번호를 즉시 마스킹 처리합니다.

03

목적별 sLLM 전용 파인튜닝

행정 업무의 각 단계와 성격에 특화된 4가지 전문 튜닝 모델이
어플라이언스에 최적화되어 탑재됩니다.

Style FT

행정 문체 특화

공문서 표준 서술 방식과 행정 용어를 완벽히 습득

Table FT

HWP 표 설계

복잡한 데이터 구조를 HWP 표 형식으로 변환 최적화

Reasoning FT

법률 추론

조문 간 상충 관계 분석 및 논리적 결론 도출

Efficiency FT

경량화 최적화

성능은 유지하면서 추론 속도와 리소스 효율 극대화

ORCHESTRA
Planner

사용자 의도 분석 및 업무 수행 로직 기획

Compliance

내부 지침 및 보안 규정 실시간 교차 검증

Style Guard

공공언어 표준 준수 및 비문 수정 에이전트

HWP Expert

HWP 컨트롤 엔진을 통한 서식 정밀 구현

Process 04

멀티 에이전트 오케스트레이션

단일 LLM의 한계를 넘어, 전문 영역별로 특화된 다수의 에이전트가 협력합니다. 기획부터 법규 검토, 최종 서식 렌더링까지 지능형 워크플로우를 완성합니다.

  • 환각(Hallucination) 방지를 위한 자가 검증 루프 탑재
  • 에이전트 간 역할 분담을 통한 초정밀 결과 도출
Interface 05

인텔리전트 에디팅 워크스페이스

지능형 근거 역추적

AI 답변의 원천이 된 법령 본문이나 과거 문서를 하이라이트와 함께 즉시 확인하여 업무 신뢰도를 확보합니다.

인터랙티브 문서 구조화

대규모 보고서의 목차를 AI가 자동 구성하고, 드래그 앤 드롭으로 문단 구조와 내용을 지능적으로 편집합니다.

의미 기반 변경 이력

단순 오타 수정을 넘어 핵심 내용의 변화와 문맥적 차이를 요약 보고하여 의사결정 속도를 2배 이상 높입니다.

Management 06

관리자 관측성 및
전방위 보안 제어

1
로컬 프롬프트 스튜디오

외부 유출 없이 내부망에서 부서별 페르소나와 최적 프롬프트를 배포·관리합니다.

2
Self-hosted 통합 모니터링

모든 토큰 사용량과 AI 성능 지표를 외부 전송 없이 내부 서버에서 독립적으로 관리합니다.

3
RBAC 기반 권한 통제

직무 그룹별로 접근 가능한 데이터 범위와 에이전트 기능을 중앙에서 엄격히 차단합니다.

Infrastructure 07

어플라이언스
특화 아키텍처

국가 보안 기관을 위한 에어갭(Air-gap) 환경을 완벽히 지원하며, 소프트웨어와 하드웨어가 결합된 고성능 인프라를 제공합니다.

Secure Gate Update

물리적 폐쇄망 환경에서도 전용 게이트웨이를 통해 최신 모델과 지식 베이스를 안전하게 업데이트합니다.

HWP 가속 엔진

하드웨어 단에서 HWP 가속 서버를 탑재하여 대량의 복합 문서 출력을 실시간으로 처리합니다.

vLLM GPU 분할

가상 GPU 기술을 적용하여 동시 사용자에게 추론 자원을 지능적으로 배분하고 지연을 최소화합니다.

영구 보존 표준

생성된 모든 산출물을 국가기록원 PDF/A 표준에 따라 자동 보관하여 행정 효력을 보장합니다.
