파트너와 고객을 연결하여 혁신적인 비즈니스 생태계를 구축하고 새로운 수익(Monetization) 을 창출하세요.

CORE FEATURES

API 라이프사이클의 모든 것을 통제하다

기업의 디지털 트랜스포메이션을 가속화하는 핵심 전략 도구

철벽 방어, 다층적 API 보안
(Security)

기업의 소중한 데이터 자산을 강력하게 보호합니다. API Key, OAuth 2.0, JWT 기반의 철저한 인증/인가 처리는 물론, 정책·엔드포인트 매핑을 통해 세분화된 룰(정규식 등)을 적용합니다. 또한 데이터 타입별 민감 정보(주민번호, 카드번호) 마스킹 기능으로 컴플라이언스를 완벽히 준수합니다.

사각지대 없는 초정밀 모니터링
(Analytics)

Redis와 Kafka로 적재되는 데이터를 실시간으로 조회하여 API 성능과 가용성을 추적합니다. 호출 추이, Top N (호출량/보안이슈), 국가별 통계를 대시보드에서 직관적으로 파악할 수 있으며, 이상 징후 발생 시 즉각적인 SMTP 알림 경고를 통해 장애를 선제적으로 예방합니다.

개발자 친화적 포털 및 협업
(Developer Portal)

API 중심 생태계의 핵심은 '개발자 경험'입니다. 직관적인 API 명세 조회(Smart Doc), 즉각적인 API 샌드박스 테스트 환경(요청 헤더/파라미터 조작), Third-Party 연동을 위한 SDK 제공을 통해 내/외부 개발자가 혁신적인 비즈니스 모델을 신속하게 런칭하도록 돕습니다.

SI 환경에 최적화된 맞춤형 확장성
(Customization)

표준화된 솔루션의 한계를 넘어섭니다. 고객의 고유한 비즈니스 요건에 맞춰 커스텀 정책(호출 제한, 특정 룰 등)을 추가 개발할 수 있으며, 플러그인 기능은 제화하는 모듈형 아키텍처로 비용 효율성과 SI 프로젝트의 빠른 배포 속도를 완벽하게 보장합니다.

PHYSICAL ARCHITECTURE

대규모 트래픽을 지탱하는 압도적 기술력

시스템 부하 없이 실시간 데이터를 캡처하고 분석하는 고가용성(HA) 아키텍처

1. 데이터 캡처 (eBPF)

Packet Beats 에이전트를 통해 시스템 커널 스페이스에서 SSL/TLS로 암호화된 트래픽을 복호화하여 패킷 데이터를 직접 캡처합니다. 기존 프록시 방식 대비 비기적으로 시스템 부하를 최소화합니다.

2. 비동기 처리 & 앤진

수집된 패킷은 고성능 네트워크 프레임워크인 Netty를 통해 비동기로 수신됩니다. 데이터는 즉시 Memory DB에 적재되며, 최적화된 Detection Engine이 실시간으로 보안 룰 위반 여부를 분석합니다.

3. 하이브리드 스토리지

용도에 맞춘 최적의 저장소를 구축했습니다.

  • Redis: 캐싱 및 실시간 통계 데이터 처리
  • OpenSearch: Raw 데이터 통계 및 비시간적 검색
  • MariaDB: 정책, 룰, 메타데이터의 장기 보관 (감사)

엔터프라이즈 어플라이언스 및 고가용성(HA) 지원

Active-Standby 이중화 및 DRDB를 이용한 무중단 서비스 인프라 구성

자주 묻는 질문 (FAQ)

SyncAPIM은 어떤 솔루션인가요?

Sync Series는 SyncCMS, SyncAny 등 다양한 디지털 솔루션을 통합 제공하는 플랫폼입니다. 콘텐츠 관리부터 파일 동기화까지 비즈니스에 필요한 모든 기능을 하나의 생태계에서 제공합니다.

기존 레거시 API게이트웨이와 비교할 때 장점은 무엇인가요?

SyncAny는 오픈 소스 기반의 클라우드 스토리지 및 파일 공유 동기화 애플리케이션입니다. 다양한 환경에서 안전하고 빠른 파일 동기화를 지원합니다.

Sync 시리즈의 다른 제품과 어떻게 연동되나요?

SyncAdmin은 전체 Sync Series 플랫폼의 통합 관리 대시보드입니다. 사용자 관리, 권한 설정, 모니터링, 로그 분석 등 운영에 필요한 모든 기능을 제공합니다.

개인정보보호와 관련된 보안 규정(컴플라이언스)을 준수하나요?

SyncCMS는 표준 RESTful API를 완벽하게 지원합니다. API 키 발급 후 원하는 프론트엔드 또는 모바일 앱과 자유롭게 연동할 수 있으며, Swagger UI로 모든 API 명세를 확인할 수 있습니다.
